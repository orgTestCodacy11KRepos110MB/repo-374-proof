Why You'll Love Proof

Easy to Use, and Extend

Built for Modern Web Apps

Proof uses JWT, the most robust and secure authentication system possible for Single-Page Applications. With Proof, authenticating users in your Ember, Angular, or Backbone app is no longer a chore. Simply install proof and the corresponding client library, and your authentication system will be up and running.

# authentication_controller.rb
class AuthenticationController < ApplicationController
  proof_actions
  before_action :require_proof, except: :login
end
# application_controller
class ApplicationController < ActionController::Base
  before_action :require_proof
  def secure_information
    render json: { secret: "I'm a Dolphin" }
  end
end
# routes.rb
Rails.application.routes.draw do
  get '/login' => 'authentication#login'
end

Stunningly Simple

Setting up proof is as easy as installing the Gem and adding two lines of code to your Rails app. There are already libraries that work with Proof for all of the popular front-end frameworks. Proof integrates with every aspect of the Rails tool chain: it comes with Helpers and Generators which get you up and running effortlessly.

Incredibly Secure

Proof is based on JWT, or Json Web Tokens. JWT combines the ease of use of JSON with rock-solid encryption technology, making cross-origin authentication as easy to use as Cookies were back in the day. Proof is also completely transparent and open-source: you can be sure that it is safe to use, and extend it as needed!

Up and Running in Three Steps

1

Install the Gem

# Gemfile
gem 'proof-rails', require: 'proof'
2

Setup Controller

# authentication_controller
class AuthenticationController
  proof_actions
end
3

Add Login Route

# routes.rb
get '/login' => 'authentication#login'
